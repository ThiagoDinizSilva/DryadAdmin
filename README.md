## About The Project

Live Preview: https://thiagodinizsilva.github.io/DryadAdmin/

 This project idea is to create a simple interface to manage ldap databases. Allowing you to add, import, delete and change varios users or groups with a simple interface.
 To access the server and manage ldap i'll use a NPM package [ldap-client](https://www.npmjs.com/package/ldap-client)
 there is a login interface and credential management as well. maybe in the future i'll try to create a rpm package or a docker container so it can be implemented as a template
